# Preferences App

* **Öğrenci Adı:** ömer göktürk

## Proje Açıklaması

Bu proje, React Native kullanılarak geliştirilmiş bir tercihler ve kimlik doğrulama uygulamasıdır.
Uygulamada Context API ile global state yönetimi, AsyncStorage ile yerel kalıcılık ve custom hook kullanımı uygulanmıştır.

## Özellikler

* Giriş / çıkış (sahte kimlik doğrulama)
* Kalıcı giriş oturumu
* Global kullanıcı state’i (Context API)
* Kalıcı tema tercihi (açık / koyu)
* Ayarlar ekranı
* Custom hook ile tema yönetimi

## Kurulum Talimatları

1. Projeyi klonlayın:

```
git clone <github.com/gokturk7/PreferencesApp>
```

2. Proje klasörüne girin:

```
cd PreferencesApp
```

3. Bağımlılıkları yükleyin:

```
npm install
```

4. Uygulamayı başlatın:

```
npx expo start
```

5. Expo Go uygulaması ile QR kodu okutun.
